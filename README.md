# AcadFlow

Plataforma web full stack para organização acadêmica de estudantes universitários.

## Sobre o projeto

O AcadFlow é uma plataforma web voltada para estudantes universitários organizarem sua rotina acadêmica em um único painel.

O sistema permite cadastrar disciplinas, tarefas, prazos, documentos acadêmicos, atividades de estágio, TCC e editais. A proposta é ajudar o estudante a acompanhar pendências importantes de forma centralizada, evitando perda de prazos e dificuldade de organização.

## Problema

Muitos estudantes universitários precisam acompanhar disciplinas, tarefas, documentos, estágios, TCCs e editais ao mesmo tempo. Essas informações normalmente ficam espalhadas em grupos de mensagens, e-mails, sistemas acadêmicos, planilhas ou anotações pessoais.

Isso pode causar esquecimento de prazos, dificuldade de acompanhamento e falta de organização da rotina acadêmica.

## Solução

O AcadFlow centraliza as principais informações acadêmicas em um dashboard simples e organizado. O usuário poderá visualizar disciplinas, tarefas pendentes, documentos necessários e editais importantes em um único ambiente.

## Funcionalidades do MVP

- Cadastro e login de usuários;
- Dashboard acadêmico;
- Cadastro de disciplinas;
- Cadastro de tarefas e prazos;
- Checklist de documentos acadêmicos;
- Área para editais e oportunidades;
- Status das pendências;
- Integração com banco de dados PostgreSQL.
